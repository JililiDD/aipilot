## AI Pilot JL Plugin Repo

This repository is the AI Pilot JL plugin source, not a product project using the plugin.

- Do not run the AI Pilot JL document workflow inside this repo.
- Do not create or commit `docs/aipilot/` or other skill-generated workflow documents here.
- Apply workflow-rule changes directly to the plugin source files under `skills/`, `commands/`, `hooks/`, adapters, or tests as appropriate.
- If a durable repo constraint is discovered, record it in this `AGENTS.md` file rather than generating workflow documents.
