# Privacy Policy

Effective date: July 19, 2026

AIPilot is a local, skills-only plugin. The plugin does not operate a hosted
service, create user accounts, use analytics, include advertising, or transmit
data to the publisher.

## Data handled by the plugin

AIPilot instructs the host coding agent to read and update files that the user
places in scope. Depending on the requested workflow, these files can include
source code, product specifications, plans, review documents, and project
memory files. Generated review HTML is temporary and the workflow instructs
the host to delete it after the review session.

The selected host application may process prompts, files, tool output, and
telemetry under its own terms and privacy policy. AIPilot does not control the
host application's data handling.

## Network access and third parties

The plugin does not define an MCP server, connector, account integration, or
publisher-controlled network endpoint. It includes vendored copies of ezreview
and marked so its document-review renderer can operate without downloading
runtime packages. Their licenses are included with the distributed files.

## Storage and deletion

Project documents created by AIPilot remain in locations selected or approved
by the user. Users can delete those files at any time. Uninstalling the plugin
removes the plugin package according to the host application's normal plugin
management behavior, but does not automatically delete project documents.

## Changes

Material changes to this policy will be recorded in the repository and shipped
with a new plugin release.

## Contact
